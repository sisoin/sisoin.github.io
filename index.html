About
소개

분산 시스템과 AI 에이전트가 만나는 지점에서 일하는 것을 좋아합니다. Kubernetes AIOps 플랫폼 OnePO에서는 Spring Boot 백엔드와 Vue 프론트엔드, 그리고 장애를 진단·복구하는 LLM 에이전트까지 여러 계층을 넘나들며 기능을 구현했습니다.

MSA 환경에서는 15개 이상의 마이크로서비스를 Helm·ArgoCD GitOps로 배포하고 Kafka로 서비스 간 이벤트를 동기화했으며, AI 영역에서는 LangGraph Supervisor 패턴으로 멀티에이전트 파이프라인을 설계해 RAG·웹서치 기반 자동 분석 보고서를 만들어 왔습니다.

"추측하지 않고 확인한 뒤 진행한다"는 원칙으로, 요구사항과 아키텍처를 먼저 이해하고 읽기 좋은 코드로 옮기는 과정을 중요하게 생각합니다.
